# Assets Required

This project requires the following media assets to be placed in this directory:

## Required Files:
1. **hero.mp4** - A hero video for the landing page (birds/wetland footage)
2. **painted_stork_gen.png** - Image of a painted stork
3. **pelican_gen.png** - Image of a spot-billed pelican
4. **village_birds_gen.png** - Image of birds in the village setting

## Where to get them:
- Download from the original repository: https://github.com/studevkiran/kokkre_bellur/tree/main/src/assets
- Or use your own images/videos of Kokkare Bellur wetland and birds

## Temporary Solution:
For now, the app will work but these images won't display until you add the actual files.
You can use placeholder images or download them from the GitHub repository.

## Also needed in public folder:
- **favicon.png** - Website favicon (place in /public folder)
